기기 확인 중

울림앱을 설치하시겠습니까?

홈 화면에 울림앱을 추가하면
출석, 공지, 연습실 예약을 더 빠르게 사용할 수 있습니다.

안드로이드 설치 방법
1. 울림앱 설치하기 버튼을 누르세요.
2. 설치창이 뜨면 설치를 누르세요.
3. 홈 화면에 울림앱 아이콘이 생성됩니다.

설치 버튼이 안 뜨면 브라우저 메뉴에서 앱 설치 또는 홈 화면에 추가를 눌러주세요.
아이폰 Safari 설치 방법
1. Safari 하단의 공유 버튼을 누르세요.
2. 홈 화면에 추가를 선택하세요.
3. 오른쪽 위 추가를 누르세요.
4. 홈 화면에 울림앱 아이콘이 생성됩니다.
아이폰 Chrome 설치 방법
1. 주소창 옆 또는 하단의 공유 버튼을 누르세요.
2. 홈 화면에 추가를 선택하세요.
3. 추가를 누르세요.

아이폰에서는 자동 설치 팝업 호출이 제한됩니다. 가장 안정적인 방식은 Safari에서 홈 화면에 추가하는 것입니다.
이미 앱 모드로 실행 중입니다.
현재 울림앱이 홈 화면 앱 형태로 열려 있습니다.
