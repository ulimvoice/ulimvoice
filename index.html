전체 공지 / 결석 알림톡 발송

반을 선택하거나 검색한 뒤 학생을 불러오고, 원하는 학생만 체크해서 공지를 발송합니다.

수신 대상 발송 방식
공지 대상 0명

출석부 확인 / 출결 발송

수업일 선택->수업명 선택 똑는 검색-> 출석부 반영 버튼 클릭

수신 대상
발송 방식
알림톡은 승인된 템플릿 ID가 필요하고, 문자는 SOLAPI 발신번호 등록이 필요합니다.
강사 권한은 발송이 제한됩니다. 출석 확인과 일일평가 저장만 가능합니다.

수강생 수업 일일평가

학생의 일일평가 작성. 수업일 선택-> 반명 선택 또는 검색-> 현재 반 학생 불러오기-> 작성 완료후 평가저장

사용 가능 치환어: [학생명], [반명], [반이름], [출석여부], [수업일], [담당강사], [수업영상링크]
평가 수신 대상
발송 방식
일일평가 알림톡은 승인된 평가 템플릿이 필요합니다.

발성 연습일지 체크

날짜별 발성훈련 기록을 확인하고, 녹음파일과 AI 코멘트를 보며 학생에게 남길 강사 코멘트를 작성합니다.

조회 전

강의실 사용체크

강의실별 사용 현황을 시간표 형태로 확인하고, 현재 사용 중인 강의실을 체크합니다.
같은 날짜·같은 시간·같은 강의실은 중복 체크되지 않습니다.

조회 전 초록: 사용중 빈칸 클릭: 해당 시간 바로 체크
※ 사용 해제는 해당 칸의 [해제] 버튼을 누르면 됩니다. 해제 시 출석부 안의 강의실 사용일지 해당 칸이 비워집니다.

연습실 예약 알림톡

예약한 학생 명단을 확인한 뒤 연습실 사용확인, 연습실 사용불가, 장소·시간 변경 알림톡을 선택 발송합니다.
현관 비밀번호는 연습실 예약 스프레드시트의 연습실설정 시트 또는 GAS 스크립트 속성값을 발송 시점에 참조합니다.

수신 대상
기본은 학생에게 발송합니다. 학부모에게도 보내려면 학부모를 체크하세요.

발송 방식

현재 수정본은 결제 내역을 웹결제기록 시트에 저장하고 결제 확인 메시지를 발송합니다. 수입최종 시트 자동 체크는 수입최종 스프레드시트 ID와 열 구조가 확정되면 바로 연결하면 됩니다.
